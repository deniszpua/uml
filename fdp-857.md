# uml
UML Diagrams 

Scripts for generating UML diagrams via *PlantUML* proxy. E.g.
```
![client-networking-scheme](http://www.plantuml.com/plantuml/proxy?src=https://raw.githubusercontent.com/deniszpua/uml/master/fdp-645.txt?version=1)
```
will result in 


![client-networking-scheme](http://www.plantuml.com/plantuml/proxy?src=https://raw.githubusercontent.com/deniszpua/uml/master/fdp-645.txt?version=1)
